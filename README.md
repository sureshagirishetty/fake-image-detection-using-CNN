# fake-image-detection-using-CNN
• Situation: On Socail Meadia Digitally manipulated images were increasingly missused,to tackel this problem, requiring automated detection at scale.
• Task: Built a binary image classifier to distinguish authentic from manipulated images using deep learning.
• Action: Developed and trained a CNN model using Python, TensorFlow/Keras, and OpenCV on UC Merced Land Use & SUN Scene datasets; applied data augmentation to improve generalization.
• Result: Achieved reliable classification accuracy, demonstrating a complete ML development lifecycle from data preprocessing to model evaluation.
